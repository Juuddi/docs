### replaceableTransaction()

> **replaceableTransaction**(`startBlock`): `QuaiTransactionResponse`

Returns a new TransactionResponse instance which has the ability to
 detect (and throw an error) if the transaction is replaced, which
 will begin scanning at `startBlock`.

 This should generally not be used by developers and is intended
 primarily for internal use. Setting an incorrect `startBlock` can
 have devastating performance consequences if used incorrectly.

### wait()

> **wait**(`confirms`?): `Promise`\<`null` \| [`ContractTransactionReceipt`](ContractTransactionReceipt.md)\>

Resolves once this transaction has been mined and has
 `confirms` blocks including it (default: `1`) with an
 optional `timeout`.

 This can resolve to `null` only if `confirms` is `0`
 and the transaction has not been mined, otherwise this will
 wait until enough confirmations have completed.

#### Parameters

• **confirms?**: `number`

The number of confirmations to wait for.

#### Returns

`Promise`\<`null` \| [`ContractTransactionReceipt`](ContractTransactionReceipt.md)\>

The transaction receipt, or `null` if `confirms` is `0`.

`QuaiTransactionResponse.wait`
