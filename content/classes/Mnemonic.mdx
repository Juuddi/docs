# Class: Mnemonic

A **Mnemonic** wraps all properties required to compute [BIP-39](https://en.bitcoin.it/wiki/BIP_0039)
 seeds and convert between phrases and entropy.

### isValidMnemonic()

> `static` **isValidMnemonic**(`phrase`, `wordlist`?): `boolean`

Returns true if `phrase` is a valid [BIP-39](https://en.bitcoin.it/wiki/BIP_0039) phrase.

 This checks all the provided words belong to the `wordlist`,
 that the length is valid and the checksum is correct.

#### Parameters

• **phrase**: `string`

The mnemonic phrase.

• **wordlist?**: `null` \| [`Wordlist`](Wordlist.md)

The wordlist for the mnemonic.

#### Returns

`boolean`

True if the phrase is valid.

#### Throws

If the phrase is invalid.
