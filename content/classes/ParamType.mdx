## Methods

### format()

```ts
format(format?): string
```

Return a string representation of this type.

For example,

`sighash" => "(uint256,address)"`

`"minimal" => "tuple(uint256,address) indexed"`

`"full" => "tuple(uint256 foo, address bar) indexed baz"`

