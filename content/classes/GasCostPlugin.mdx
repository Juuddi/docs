# Class: GasCostPlugin

A **GasCostPlugin** allows a network to provide alternative values when
 computing the intrinsic gas required for a transaction.

## Extends

- [`NetworkPlugin`](NetworkPlugin.md)

## Implements

- [`GasCostParameters`](../type-aliases/GasCostParameters.md)

## Constructors

### new GasCostPlugin()

> **new GasCostPlugin**(`effectiveBlock`?, `costs`?): [`GasCostPlugin`](GasCostPlugin.md)

Creates a new GasCostPlugin from `effectiveBlock` until the
 latest block or another GasCostPlugin supercedes that block number,
 with the associated `costs`.

#### Parameters

• **effectiveBlock?**: `number`

• **costs?**: [`GasCostParameters`](../type-aliases/GasCostParameters.md)

### effectiveBlock

> `readonly` **effectiveBlock**: `number`

The block number to treat these values as valid from.

 This allows a hardfork to have updated values included as well as
 mulutiple hardforks to be supported.
