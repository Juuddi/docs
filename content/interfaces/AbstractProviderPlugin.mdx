---
title: AbstractProviderPlugin
---

An **AbstractPlugin** is used to provide additional internal services to an
[**AbstractProvider**](/content/classes/AbstractProvider) without adding backwards-incompatible changes to method signatures or
other internal and complex logic.

## Properties

| Property | Modifier | Type | Description |
| :------ | :------ | :------ | :------ |
| `name` | `readonly` | `string` | The reverse domain notation of the plugin. |

## Methods

### connect()

```ts
connect(provider): AbstractProviderPlugin
```

Creates a new instance of the plugin, connected to `provider`.

#### Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `provider` | [`AbstractProvider`](/content/classes/AbstractProvider)\<`any`\> | The provider to connect to. |
