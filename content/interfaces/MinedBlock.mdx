An Interface to indicate a [**Block**](/content/classes/Block.mdx) has been included in the
 blockchain. This asserts a Type Guard that necessary properties
 are non-null.

 Before a block is included, it is a //pending// block.

- [`Block`](/content/classes/Block.mdx)
