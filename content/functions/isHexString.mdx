# Function: isHexString()

> **isHexString**(`value`, `length`?): value is \`0x$\{string\}\`

Returns true if `value` is a valid HexString | **HexString**.

 If `length` is `true` or a //number//, it also checks that
 `value` is a valid DataHexString | **DataHexString** of `length` (if a //number//)
 bytes of data (e.g. `0x1234` is 2 bytes).

## Parameters

• **value**: `any`

The value to check.

• **length?**: `number` \| `boolean`

The expected length of the data.

## Returns

value is \`0x$\{string\}\`

True if the value is a valid HexString | **HexString**.
