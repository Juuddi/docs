# Function: checkResultErrors()

> **checkResultErrors**(`result`): `object`[]

Returns all errors found in a [**Result**](../classes/Result.md).

 Since certain errors encountered when creating a [**Result**](../classes/Result.md) do
 not impact the ability to continue parsing data, they are
 deferred until they are actually accessed. Hence a faulty string
 in an Event that is never used does not impact the program flow.

 However, sometimes it may be useful to access, identify or
 validate correctness of a [**Result**](../classes/Result.md).

## Parameters

• **result**: [`Result`](../classes/Result.md)

The Result to check for errors.

## Returns

`object`[]

An array of objects with the path to the error and the error itself.
