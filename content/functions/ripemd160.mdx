# Function: ripemd160()

> **ripemd160**(`_data`): `string`

Compute the cryptographic RIPEMD-160 hash of `data`.

## Parameters

• **\_data**: [`BytesLike`](../type-aliases/BytesLike.md)

The data to hash.

## Returns

`string`

DataHexstring

The hash of the data.

## Example

```ts
   ripemd160("0x")
   //_result:

   ripemd160("0x1337")
   //_result:

   ripemd160(new Uint8Array([ 0x13, 0x37 ]))
   //_result:
 ```
