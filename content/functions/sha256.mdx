# Function: sha256()

> **sha256**(`_data`): `string`

Compute the cryptographic SHA2-256 hash of `data`.

## Parameters

• **\_data**: [`BytesLike`](../type-aliases/BytesLike.md)

The data to hash.

## Returns

`string`

The hash of the data.

## Example

```ts
   sha256("0x")
   //_result:

   sha256("0x1337")
   //_result:

   sha256(new Uint8Array([ 0x13, 0x37 ]))
   //_result:
 ```
