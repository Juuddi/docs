# Function: toUtf8String()

> **toUtf8String**(`bytes`, `onError`?): `string`

Returns the string represented by the UTF-8 data `bytes`.

 When `onError` function is specified, it is called on UTF-8
 errors allowing recovery using the [**Utf8ErrorFunc**](../type-aliases/Utf8ErrorFunc.md) API.
 (default: [error](Utf8ErrorFuncs))

## Parameters

• **bytes**: [`BytesLike`](../type-aliases/BytesLike.md)

The UTF-8 data to convert.

• **onError?**: [`Utf8ErrorFunc`](../type-aliases/Utf8ErrorFunc.md)

The error handling function.

## Returns

`string`

The string.
