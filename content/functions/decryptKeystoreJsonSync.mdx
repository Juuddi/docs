```ts
function decryptKeystoreJsonSync(json, _password): KeystoreAccount
```

Returns the account details for the JSON Keystore Wallet `json`
 using `password`.

 It is preferred to use the [**async version**](/content/functions/decryptKeystoreJson.mdx)
 instead, which allows a [**ProgressCallback](/content/type-aliases/ProgressCallback.mdx) to keep the user informed
 as to the decryption status.

 This method will block the event loop (freezing all UI) until decryption
 is complete, which can take quite some time, depending on the wallet
 paramters and platform.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `json` | `string` | The JSON data to decrypt. |
| `_password` | `string` \| `Uint8Array` | The password to decrypt the JSON data. |

## Returns

[`KeystoreAccount`](/content/type-aliases/KeystoreAccount.mdx)

The decrypted account.
