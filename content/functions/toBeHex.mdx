```ts
function toBeHex(_value, _width?): string
```

Converts `value` to a Big Endian hexstring, optionally padded to
 `width` bytes.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `_value` | [`BigNumberish`](/content/type-aliases/BigNumberish.mdx) | The value to convert. |
| `_width`? | [`Numeric`](/content/type-aliases/Numeric.mdx) | The width of the value in bytes. |

## Returns

`string`

The hexstring.

## Throws

If the value exceeds the width.
