# Function: toBeHex()

> **toBeHex**(`_value`, `_width`?): `string`

Converts `value` to a Big Endian hexstring, optionally padded to
 `width` bytes.

## Parameters

• **\_value**: [`BigNumberish`](../type-aliases/BigNumberish.md)

The value to convert.

• **\_width?**: [`Numeric`](../type-aliases/Numeric.md)

The width of the value in bytes.

## Returns

`string`

The hexstring.

## Throws

If the value exceeds the width.
