```ts
function recoverAddress(digest, signature): string
```

Returns the recovered address for the private key that was
 used to sign `digest` that resulted in `signature`.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `digest` | [`BytesLike`](/content/type-aliases/BytesLike.mdx) | The digest of the message. |
| `signature` | [`SignatureLike`](/content/type-aliases/SignatureLike.mdx) | The signature. |

## Returns

`string`

The address.
