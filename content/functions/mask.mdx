```ts
function mask(_value, _bits): bigint
```

Mask `value` with a bitmask of `bits` ones.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `_value` | [`BigNumberish`](/content/type-aliases/BigNumberish) | The value to mask. |
| `_bits` | [`Numeric`](/content/type-aliases/Numeric) | The number of bits to mask. |

## Returns

`bigint`

The masked value.
