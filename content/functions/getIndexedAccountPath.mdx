```ts
function getIndexedAccountPath(_index): string
```

Returns the path using an alternative pattern for deriving accounts,
 at `index`.

 This derivation path uses the //index// component rather than the
 //account// component to derive sequential accounts.

 This is the pattern used by wallets like MetaMask.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `_index` | [`Numeric`](/content/type-aliases/Numeric.mdx) | The account index. |

## Returns

`string`

The BIP44 derivation path for the specified account.
