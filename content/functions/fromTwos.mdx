# Function: fromTwos()

> **fromTwos**(`_value`, `_width`): `bigint`

Convert `value` from a twos-compliment representation of `width`
 bits to its value.

 If the highest bit is `1`, the result will be negative.

## Parameters

• **\_value**: [`BigNumberish`](../type-aliases/BigNumberish.md)

The value to convert.

• **\_width**: [`Numeric`](../type-aliases/Numeric.md)

The width of the value in bits.

## Returns

`bigint`

The value.

## Throws

If the value is too large for the width.
