```ts
function assertArgument(
   check, 
   message, 
   name, 
   value): asserts check
```

A simple helper to simply ensuring provided arguments match expected constraints, throwing if not.

In TypeScript environments, the `check` has been asserted true, so any further code does not need additional
compile-time checks.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `check` | `unknown` | The value to check. |
| `message` | `string` | The error message. |
| `name` | `string` | The name of the argument. |
| `value` | `unknown` | The value of the argument. |

## Returns

`asserts check`

## Throws

Throws if `check` is falsish.
