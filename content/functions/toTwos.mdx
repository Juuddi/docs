# Function: toTwos()

> **toTwos**(`_value`, `_width`): `bigint`

Convert `value` to a twos-compliment representation of
 `width` bits.

 The result will always be positive.

## Parameters

• **\_value**: [`BigNumberish`](../type-aliases/BigNumberish.md)

The value to convert.

• **\_width**: [`Numeric`](../type-aliases/Numeric.md)

The width of the value in bits.

## Returns

`bigint`

The value.

## Throws

If the value is too large for the width.
