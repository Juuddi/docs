```ts
function formatUnits(value, unit?): string
```

Converts `value` into a //decimal string//, assuming `unit` decimal
 places. The `unit` may be the number of decimal places or the name of
 a unit (e.g. `"gwei"` for 9 decimal places).

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `value` | [`BigNumberish`](/content/type-aliases/BigNumberish.mdx) | The value to convert. |
| `unit`? | `string` \| [`Numeric`](/content/type-aliases/Numeric.mdx) | The unit to convert to. |

## Returns

`string`

The converted value.

## Throws

If the unit is invalid.
