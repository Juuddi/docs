```ts
function decodeBytes32String(_bytes): string
```

Decodes a Bytes32-encoded string into a regular string.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `_bytes` | [`BytesLike`](/content/type-aliases/BytesLike.mdx) | The Bytes32-encoded data. |

## Returns

`string`

The decoded string.

## Throws

If the input is not exactly 32 bytes long or lacks a null terminator.
