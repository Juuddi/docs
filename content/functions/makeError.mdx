# Function: makeError()

> **makeError**\<`K`, `T`\>(`message`, `code`, `info`?): `T`

Returns a new Error configured to the format quais emits errors, with
 the `message`, [**ErrorCode**](../type-aliases/ErrorCode.md) `code` and additional properties
 for the corresponding quaisError.

 Each error in quais includes the version of quais, a
 machine-readable [**ErrorCode**](../type-aliases/ErrorCode.md), and depending on `code`, additional
 required properties. The error message will also include the `message`,
 quais version, `code` and all additional properties, serialized.
