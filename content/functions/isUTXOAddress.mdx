```ts
function isUTXOAddress(address): boolean
```

Checks whether a given blockchain address is a UTXO address based on the 9th bit of
 the address. This function extracts the second byte of the address and checks its
 first bit to determine the UTXO status.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `address` | `string` | The blockchain address to be analyzed, expected to start with "0x" followed by its hexadecimal representation. |

## Returns

`boolean`

True if the address is a UTXO address, false otherwise.
