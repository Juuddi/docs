# Function: toUtf8Bytes()

> **toUtf8Bytes**(`str`, `form`?): `Uint8Array`

Returns the UTF-8 byte representation of `str`.

 If `form` is specified, the string is normalized.

## Parameters

• **str**: `string`

The string to convert.

• **form?**: [`UnicodeNormalizationForm`](../type-aliases/UnicodeNormalizationForm.md)

The normalization form to use.

## Returns

`Uint8Array`

The UTF-8 byte representation.

## Throws

If the UTF-8 conversion fails.
