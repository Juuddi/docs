# Function: toBigInt()

> **toBigInt**(`value`): `bigint`

Converts `value` to a BigInt. If `value` is a Uint8Array, it
 is treated as Big Endian data.

## Parameters

• **value**: `Uint8Array` \| [`BigNumberish`](../type-aliases/BigNumberish.md)

The value to convert.

## Returns

`bigint`

The value.
