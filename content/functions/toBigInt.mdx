title: toBigInt
---

```ts
function toBigInt(value): bigint
```

Converts `value` to a BigInt. If `value` is a Uint8Array, it is treated as Big Endian data.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `value` | `Uint8Array` \| [`BigNumberish`](/content/type-aliases/BigNumberish) | The value to convert. |

## Returns

`bigint`

The value.
