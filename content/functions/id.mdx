# Function: id()

> **id**(`value`): `string`

A simple hashing function which operates on UTF-8 strings to
 compute an 32-byte identifier.

 This simply computes the [**UTF-8 bytes**](toUtf8Bytes.md) 
 and computes the [**keccak256**](keccak256.md).

## Parameters

• **value**: `string`

The string to hash.

## Returns

`string`

The 32-byte identifier.

## Example

```ts
   id("hello world")
   //_result:
 ```
