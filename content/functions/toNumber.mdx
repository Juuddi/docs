```ts
function toNumber(value): number
```

Converts `value` to a number. If `value` is a Uint8Array, it
 is treated as Big Endian data. Throws if the value is not safe.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `value` | `Uint8Array` \| [`BigNumberish`](/content/type-aliases/BigNumberish.mdx) | The value to convert. |

## Returns

`number`

The value.

## Throws

If the value is not safe to convert to a number.
