```ts
function dataSlice(
   data, 
   start?, 
   end?): string
```

Returns a DataHexString | **DataHexString** by slicing `data` from the `start`
 offset to the `end` offset.

 By default `start` is 0 and `end` is the length of `data`.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `data` | [`BytesLike`](/content/type-aliases/BytesLike.mdx) | The data to slice. |
| `start`? | `number` | The start offset. |
| `end`? | `number` | The end offset. |

## Returns

`string`

The sliced data.
