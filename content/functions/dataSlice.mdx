# Function: dataSlice()

> **dataSlice**(`data`, `start`?, `end`?): `string`

Returns a DataHexString | **DataHexString** by slicing `data` from the `start`
 offset to the `end` offset.

 By default `start` is 0 and `end` is the length of `data`.

## Parameters

• **data**: [`BytesLike`](../type-aliases/BytesLike.md)

The data to slice.

• **start?**: `number`

The start offset.

• **end?**: `number`

The end offset.

## Returns

`string`

The sliced data.
