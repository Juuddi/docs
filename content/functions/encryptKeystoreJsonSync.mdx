# Function: encryptKeystoreJsonSync()

> **encryptKeystoreJsonSync**(`account`, `password`, `options`?): `string`

Return the JSON Keystore Wallet for `account` encrypted with
 `password`.

 The `options` can be used to tune the password-based key
 derivation function parameters, explicitly set the random values
 used. Any provided [**ProgressCallback**](../type-aliases/ProgressCallback.md) is ignored.

## Parameters

• **account**: [`KeystoreAccount`](../type-aliases/KeystoreAccount.md)

The account to encrypt.

• **password**: `string` \| `Uint8Array`

The password to encrypt the JSON data.

• **options?**: [`EncryptOptions`](../type-aliases/EncryptOptions.md)

The options to use when encrypting the JSON data.

## Returns

`string`

The encrypted JSON data.
