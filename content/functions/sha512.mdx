# Function: sha512()

> **sha512**(`_data`): `string`

Compute the cryptographic SHA2-512 hash of `data`.

## Parameters

• **\_data**: [`BytesLike`](../type-aliases/BytesLike.md)

The data to hash.

## Returns

`string`

The hash of the data.

## Example

```ts
   sha512("0x")
   //_result:

   sha512("0x1337")
   //_result:

   sha512(new Uint8Array([ 0x13, 0x37 ]))
   //_result:
 ```
