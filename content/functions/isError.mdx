# Function: isError()

> **isError**\<`K`, `T`\>(`error`, `code`): `error is T`

Returns true if the `error` matches an error thrown by quais
 that matches the error `code`.

 In TypeScript environments, this can be used to check that `error`
 matches an quaisError type, which means the expected properties will
 be set.

• **K** *extends* [`ErrorCode`](../type-aliases/ErrorCode.md)

• **T** *extends* [`UnknownError`](../interfaces/UnknownError.md) \| [`InvalidArgumentError`](../interfaces/InvalidArgumentError.md) \| [`BadDataError`](../interfaces/BadDataError.md) \| [`NotImplementedError`](../interfaces/NotImplementedError.md) \| [`UnsupportedOperationError`](../interfaces/UnsupportedOperationError.md) \| [`NetworkError`](../interfaces/NetworkError.md) \| [`ServerError`](../interfaces/ServerError.md) \| [`TimeoutError`](../interfaces/TimeoutError.md) \| [`CancelledError`](../interfaces/CancelledError.md) \| [`BufferOverrunError`](../interfaces/BufferOverrunError.md) \| [`NumericFaultError`](../interfaces/NumericFaultError.md) \| [`MissingArgumentError`](../interfaces/MissingArgumentError.md) \| [`UnexpectedArgumentError`](../interfaces/UnexpectedArgumentError.md) \| [`CallExceptionError`](../interfaces/CallExceptionError.md) \| [`InsufficientFundsError`](../interfaces/InsufficientFundsError.md) \| [`NonceExpiredError`](../interfaces/NonceExpiredError.md) \| [`ReplacementUnderpricedError`](../interfaces/ReplacementUnderpricedError.md) \| [`TransactionReplacedError`](../interfaces/TransactionReplacedError.md) \| [`ActionRejectedError`](../interfaces/ActionRejectedError.md)
