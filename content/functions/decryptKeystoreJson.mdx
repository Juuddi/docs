# Function: decryptKeystoreJson()

> **decryptKeystoreJson**(`json`, `_password`, `progress`?): `Promise`\<[`KeystoreAccount`](../type-aliases/KeystoreAccount.md)\>

Resolves to the decrypted JSON Keystore Wallet `json` using the
 `password`.

 If provided, `progress` will be called periodically during the
 decrpytion to provide feedback, and if the function returns
 `false` will halt decryption.

 The `progressCallback` will **always** receive `0` before
 decryption begins and `1` when complete.

## Parameters

• **json**: `string`

The JSON data to decrypt.

• **\_password**: `string` \| `Uint8Array`

The password to decrypt the JSON data.

• **progress?**: [`ProgressCallback`](../type-aliases/ProgressCallback.md)

The callback to receive progress updates.

## Returns

`Promise`\<[`KeystoreAccount`](../type-aliases/KeystoreAccount.md)\>

The decrypted account.
