# Function: getShardForAddress()

> **getShardForAddress**(`address`): `object` \| `null`

Retrieves the shard information for a given address based on its byte prefix.
 The function parses the address to extract its byte prefix, then filters the ShardData
 to find a matching shard entry. If no matching shard is found, it returns null.

## Parameters

• **address**: `string`

The blockchain address to be analyzed. The address should start with "0x" followed by the hexadecimal representation.

## Returns

`object` \| `null`

An object containing the shard information, or null if no
