# Function: isBytesLike()

> **isBytesLike**(`value`): `value is BytesLike`

Returns true if `value` is a valid representation of arbitrary
 data (i.e. a valid DataHexString | **DataHexString** or a Uint8Array).

## Parameters

• **value**: `any`

The value to check.

## Returns

`value is BytesLike`

True if the value is a valid DataHexString | **DataHexString**.
