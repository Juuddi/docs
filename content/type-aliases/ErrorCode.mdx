All errors emitted by quais have an **ErrorCode** to help
 identify and coalesce errors to simplify programmatic analysis.

 Each **ErrorCode** is the `code` proerty of a coresponding
[**quaisError**](/content/interfaces/quaisError.mdx).

 **Generic Errors**

 **`"UNKNOWN_ERROR"`** - see [**UnknownError**](/content/interfaces/UnknownError.mdx)

 **`"NOT_IMPLEMENTED"`** - see [**NotImplementedError**](/content/interfaces/NotImplementedError.mdx)

 **`"UNSUPPORTED_OPERATION"`** - see [**UnsupportedOperationError**](/content/interfaces/UnsupportedOperationError.mdx)

 **`"NETWORK_ERROR"`** - see [**NetworkError**](/content/interfaces/NetworkError.mdx)

 **`"SERVER_ERROR"`** - see [**ServerError**](/content/interfaces/ServerError.mdx)

 **`"TIMEOUT"`** - see [**TimeoutError**](/content/interfaces/TimeoutError.mdx)

 **`"BAD_DATA"`** - see [**BadDataError**](/content/interfaces/BadDataError.mdx)

 **`"CANCELLED"`** - see [**CancelledError**](/content/interfaces/CancelledError.mdx)

 **Operational Errors**

 **`"BUFFER_OVERRUN"`** - see [**BufferOverrunError**](/content/interfaces/BufferOverrunError.mdx)

 **`"NUMERIC_FAULT"`** - see [**NumericFaultError**](/content/interfaces/NumericFaultError.mdx)

 **Argument Errors**

 **`"INVALID_ARGUMENT"`** - see [**InvalidArgumentError**](/content/interfaces/InvalidArgumentError.mdx)

 **`"MISSING_ARGUMENT"`** - see [**MissingArgumentError**](/content/interfaces/MissingArgumentError.mdx)

 **`"UNEXPECTED_ARGUMENT"`** - see [**UnexpectedArgumentError**](/content/interfaces/UnexpectedArgumentError.mdx)

 **`"VALUE_MISMATCH"`** - //unused//

 **Blockchain Errors**

 **`"CALL_EXCEPTION"`** - see [**CallExceptionError**](/content/interfaces/CallExceptionError.mdx)

 **`"INSUFFICIENT_FUNDS"`** - see [**InsufficientFundsError**](/content/interfaces/InsufficientFundsError.mdx)

 **`"NONCE_EXPIRED"`** - see[**NonceExpiredError**](/content/interfaces/NonceExpiredError.mdx)

 **`"REPLACEMENT_UNDERPRICED"`** - see [**ReplacementUnderpricedError**](/content/interfaces/ReplacementUnderpricedError.mdx)

 **`"TRANSACTION_REPLACED"`** - see [**TransactionReplacedError**](/content/interfaces/TransactionReplacedError.mdx)

 **`"UNCONFIGURED_NAME"`** - see {link UnconfiguredNameError | **UnconfiguredNameError**}
