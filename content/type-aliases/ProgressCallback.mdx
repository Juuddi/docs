```ts
type ProgressCallback: (percent) => void;
```

A callback during long-running operations to update any UI or provide programatic access to the progress.

The `percent` is a value between `0` and `1`.

## Parameters

| Parameter | Type |
| :------ | :------ |
| `percent` | `number` |

## Returns

`void`
